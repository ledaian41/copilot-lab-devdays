# Soc Ops: Social Bingo Game 🎉

Welcome to **Soc Ops**, the ultimate icebreaker game for in-person mixers and team-building events! Turn networking into a fun, interactive bingo experience where you find people who match quirky questions and aim for 5 in a row.

## 🌟 What is Soc Ops?

Soc Ops is a **Blazor WebAssembly** app that brings the classic bingo game to life in a social context. Players are presented with fun, engaging questions like "Find someone who has traveled to another country" or "Find someone who loves coding in their free time." Mark off squares as you connect with others, and be the first to get a bingo!

Perfect for:
- Conference networking sessions
- Team-building workshops
- Icebreaker activities at meetups
- Any gathering where people want to mingle and learn about each other

## 🚀 Quick Start

Get started in seconds! No installation required – just click the link above to play in your browser.

For developers: Clone the repo and run locally.

### Prerequisites
- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0) or higher

### Run Locally
```bash
cd SocOps
dotnet run
```

### Build
```bash
cd SocOps
dotnet build
```

> Deploys automatically to GitHub Pages on push to `main`.
